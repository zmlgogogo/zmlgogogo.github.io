温度、压力、流量、液位这类连续量的稳定控制，几乎都离不开 PID 闭环。西门子 S7-1200/1500 内置了 PID_Compact 工艺对象，把 PID 算法、自整定、量程标定封装好，配上图形化调试面板，让闭环控制变得相当简单。本文讲解 PID_Compact 的原理、组态、参数与整定调试全流程。

一、PID 闭环控制基础#

1.1 闭环控制原理#

闭环控制就是「测量 → 比较 → 调节」的不断循环：

  • 设定值（Setpoint）：你想要的目标，如 80℃。
  • 实际值（Process Value）：传感器测到的当前值。
  • 偏差（Error）= 设定值 − 实际值。
  • PID 运算：根据偏差算出**输出值（Output）**去驱动执行器（加热、阀门、变频器）。
  • 实际值变化又被测量，形成闭环。

1.2 PID 三个作用#

环节 参数 作用 调大的影响
比例 P 比例增益 Kp 偏差越大输出越大，响应主力 响应快，过大会振荡
积分 I 积分时间 Ti 消除稳态余差 Ti 越小积分越强，过强易超调振荡
微分 D 微分时间 Td 抑制变化趋势，提前刹车 改善动态，过大对噪声敏感

西门子 PID_Compact 用 Kp（增益）、Ti（积分时间，秒）、Td（微分时间，秒） 表示。Ti/Td 越大对应作用越弱（时间常数形式）。温度控制常用 PI 或 PID，流量/压力多用 PI。

二、PID_Compact 工艺对象#

PID_Compact 位于 工艺 → PID 控制 → Compact PID，是带自整定的通用 PID 控制器。

2.1 添加与调用#

  1. 在 PLC 程序中添加工艺对象 → PID_Compact，生成背景 DB。
  2. 在循环中断 OB（如 OB30/OB35） 中调用 PID_Compact，保证固定采样周期执行 —— 这是 PID 稳定的前提。

关键：PID 必须在循环中断 OB 里调用，采样周期固定（如 100ms / 1s，按被控对象快慢选）。放在 OB1 里扫描周期不固定，会影响积分微分计算。

2.2 主要引脚参数#

              PID_Compact
            ┌──────────────────────┐
Setpoint ──►│ Setpoint      Output ├──► 输出(0~100%)
  Input ───►│ Input    Output_PER  ├──► 模拟量输出值
            │          Output_PWM  ├──► PWM 输出(开关量)
            │              State   ├──► 当前模式
ManualEn ──►│ ManualEnable  Error  ├──► 错误
ManualVal─►│ ManualValue          │
            └──────────────────────┘
参数 说明
Setpoint 设定值（工程量，如 80.0℃）
Input 实际值（工程量，REAL），用经标定的过程值
Input_PER 直接接模拟量输入字（外设值），由组态做标定
Output PID 输出，0.0~100.0%（REAL）
Output_PER 模拟量输出值（接 AO）
Output_PWM PWM 脉宽输出（接开关量执行器，如固态继电器加热）
ManualEnable / ManualValue 手动模式使能 / 手动输出值
State / Mode 当前/目标工作模式（0 未激活,1 预调节,2 精确调节,3 自动,4 手动)

2.3 三种输出选择#

输出 接什么执行器
Output（百分比） 程序里再处理，或接变频器（换算后给设定值）
Output_PER 模拟量阀、模拟量调节的执行器（AO 0-10V/4-20mA）
Output_PWM 开关量执行器（固态继电器加热棒，按周期通断）

三、组态步骤#

双击 PID_Compact 的「组态」，按向导设置：

3.1 基本设置#

  • 控制器类型：选物理量与单位（温度℃、压力 bar 等），决定显示与标定。
  • 反作用：加热是正作用（输出增→实际值增）；制冷/降温场合需勾选反转控制逻辑。
  • 选择输入/输出引脚类型（Input 还是 Input_PER；Output / Output_PER / Output_PWM）。

3.2 过程值标定（量程）#

用 Input_PER 时，把模拟量原始值（0~27648 对应量程）标定为工程量：

  • 设定标定下限/上限：如传感器 027648 对应 0150℃。
  • PID 内部即用工程量（℃）运算，直观。

3.3 过程值限值#

设置实际值的上下限报警，超限可触发 Error，保护设备（如超温停机）。

3.4 输出限值#

设置输出的上下限（如 0100%，或限制 2080%），以及出错时的替代输出。

四、PID 整定（核心步骤）#

整定就是找到合适的 Kp/Ti/Td。PID_Compact 提供自整定，强烈推荐先自整定再微调。

4.1 调试面板#

打开 PID_Compact 的调试面板（在线），可实时看设定值、实际值、输出的趋势曲线，并启动整定。

4.2 预调节（Pretuning）#

  • 作用：在实际值平稳时启动，PID 主动给一个输出阶跃，观察系统响应，自动算出初步 Kp/Ti/Td。
  • 条件：实际值要相对稳定、离设定值有一定距离。
  • 适合首次整定，快速得到一组可用参数。

4.3 精确调节（Fine tuning）#

  • 作用：在实际值接近设定值时启动，PID 制造小幅振荡，精细优化参数，得到更好的动态。
  • 一般在预调节之后做，进一步提升控制品质。

4.4 整定流程#

  1. 系统接好、量程标定正确、Output 能真正驱动执行器。
  2. 调试面板启动预调节，等待完成（State 显示进度），参数自动写入。
  3. 再启动精确调节，完成后参数进一步优化。
  4. 观察趋势曲线：超调、调节时间、稳态误差是否满意。
  5. 不满意可手动微调 Kp/Ti/Td 或重做整定。

4.5 手动微调口诀#

现象 调整
响应太慢 增大 Kp（或减小 Ti）
振荡 / 超调大 减小 Kp、增大 Ti
有稳态余差消不掉 减小 Ti（加强积分）
对扰动恢复慢、想更稳 适当加 Td（微分），但噪声大时慎用

五、实战示例：温度闭环（PWM 加热）#

场景：S7-1200 控温，PT100 经模拟量输入测温，固态继电器（SSR）控制加热棒，目标 80℃。

  1. 工艺对象 PID_Compact，在 OB30（循环中断 200ms） 调用。
  2. 组态：控制器类型「温度℃」，正作用；Input_PER 接温度 AI，标定 027648 → 0150℃；输出选 Output_PWM 接 SSR。
  3. OB30 调用：
"PID_Compact_1"(Setpoint := 80.0,        // 目标 80℃
                Input_PER := "IW64");    // 温度模拟量输入
// Output_PWM 自动按占空比通断，接到 Q 点驱动 SSR
"Q0.0" := "PID_Compact_1".Output_PWM;
  1. 在线打开调试面板 → 启动预调节 → 再精确调节 → 看曲线，超调过大就重整定或减 Kp。

六、常见问题与排查#

现象 可能原因 排查方向
实际值不动 / 输出无效 输出没接对执行器、限值为 0 查 Output 接线、输出上限
控制反向（越调越偏） 作用方向设反 加热正作用、制冷勾选反转逻辑
实际值数值不对 量程标定错 核对标定上下限与传感器量程
长期消不掉余差 积分太弱或纯 P 减小 Ti，确保有积分作用
持续振荡 Kp 太大 / Ti 太小 / 采样周期不当 减 Kp、增 Ti、确认在循环中断 OB
整定失败 启动条件不满足、扰动大 实际值先稳定、远离设定值再预调节
输出忽大忽小抖动 微分对噪声敏感 减小 Td、对输入滤波

排查口诀#

  1. 先通回路：传感器测得准、执行器动得了、输出真能影响实际值。
  2. 校量程方向：标定上下限正确、作用方向对（加热正/制冷反）。
  3. 定采样：PID 放在循环中断 OB，采样周期固定且合适。
  4. 先自整定：预调节得初值 → 精确调节优化，别一上来手调。
  5. 看曲线微调：按「慢加 Kp、振荡减 Kp 增 Ti、余差减 Ti」调。

七、要点速查#

环节 关键点
调用位置 循环中断 OB，采样周期固定
输入 Input（工程量）或 Input_PER（外设值+标定）
输出 Output(%) / Output_PER(模拟量) / Output_PWM(开关量)
方向 加热正作用，制冷勾选反转逻辑
整定 预调节得初值 → 精确调节优化
参数 Kp 增益、Ti 积分时间、Td 微分时间（Ti/Td 越大作用越弱）
调试 在线趋势曲线 + 调试面板启动整定

结语#

PID_Compact 把闭环控制里最难的算法和整定都封装好了，工程里用好它，关键在三步：把回路接通并标定准（传感器、执行器、量程、方向）→ 放在循环中断里固定采样 → 先自整定再看曲线微调。理解 P 快、I 消差、D 抑振这三者的分工，配合预调节和精确调节，温度、压力、流量这些连续量的闭环控制就能又快又稳。这也是从「开关量逻辑控制」迈向「过程控制」的一项核心技能。
