西门子 G120 / V20 变频器 PROFINET 控制详解（S7-1200/1500）

讲解西门子 G120 / V20 变频器经 PROFINET 与 S7-1200/1500 通讯控制：标准报文 1/352、控制字 STW1 与状态字 ZSW1、主设定值频率换算、起停调速编程，以及 BOP/参数设置与常见故障排查。

变频器是工业现场用量最大的执行设备之一，用于风机、水泵、传送带等的调速控制。西门子 SINAMICS G120（模块化通用变频器）与 V20（紧凑经济型）都支持 PROFINET，可由 S7-1200/1500 通过总线直接起停、调速、读取状态。本文讲解 G120/V20 经 PROFINET 与 PLC 的通讯控制全流程。和前面 V90 伺服那篇正好配成「运动控制」双子篇。

一、变频器控制方式概览#

变频器可由 PLC 通过几种方式控制：

方式 说明 特点
端子控制 数字量起停 + 模拟量调速 接线多、点位有限
PROFINET / PROFIBUS 总线报文控制 接线少、功能全、可读全部状态（本文重点）
Modbus / USS 串口通讯控制 V20 常用，速率低

G120 全系支持 PROFINET（带 PN 控制单元，如 CU240E-2 PN）；V20 经 G120 不同，本体支持 USS/Modbus，PROFINET 需特定型号或扩展。本文以 G120 PROFINET 为主线，V20 思路类似（报文/控制字概念通用）。

二、报文与过程数据#

PROFINET 控制变频器的核心是周期性交换过程数据（PZD），由 PLC 写「控制字 + 设定值」、读「状态字 + 实际值」。常用报文：

报文 PLC→变频器 变频器→PLC 用途
标准报文 1 STW1 + 主设定值(NSOLL_A) ZSW1 + 主实际值(NIST_A) 基本调速，最常用
标准报文 352 STW1 + 设定值 + 6 字过程数据 ZSW1 + 实际值 + 故障/电流等 带更多监控数据
标准报文 20 同 1，西门子简化 VIK/NAMUR

报文 1 的结构（每个方向 2 个字 / 4 字节）：

方向 字 1 字 2
PLC→变频器 STW1（控制字） NSOLL_A（主设定值，速度给定）
变频器→PLC ZSW1（状态字） NIST_A（主实际值，速度反馈）

2.1 控制字 STW1（关键位）#

控制字按位控制变频器，标准报文 1 的常用值：

位 含义 说明
0 ON/OFF1 1=启动（按斜坡），0=停车（按斜坡 OFF1）
1 OFF2 0=自由停车（惯性滑行）
2 OFF3 0=快速停车
3 运行使能 1=使能运行
4 斜坡发生器使能 1=使能
5 斜坡继续 1=继续
6 设定值使能 1=设定值生效
7 故障复位 上升沿复位故障
10 由 PLC 控制 1=由 PLC 控制（必须置 1）
11 反向 1=反转

常用启动值：先送 16#047E（准备就绪，各使能位置 1、ON 位为 0），再送 16#047F（bit0 置 1 启动）。bit10「由 PLC 控制」必须为 1，否则变频器不接受总线命令。复位故障在 bit7 给上升沿。

2.2 状态字 ZSW1（关键位）#

位 含义
0 准备接通
1 准备运行
2 运行使能（电机运转中）
3 故障
7 报警
11 设定值/实际值偏差

读 ZSW1 各位可判断变频器当前是否就绪、运行、故障。

2.3 设定值 / 实际值的换算#

主设定值 NSOLL_A 是 16 位规格化值：16#4000（16384）= 100% 参考转速（参数 p2000 定义的参考值）。

设定值(hex) = 目标转速 / 参考转速 × 16384
例：参考转速 1500rpm，要 750rpm → 750/1500 × 16384 = 8192 = 16#2000
负值表示反向（补码）。

实际值 NIST_A 同理：实际转速 = NIST_A / 16384 × 参考转速。

三、变频器侧参数设置#

用 BOP-2 操作面板 或 Startdrive（TIA Portal 集成） 设置：

  1. 快速调试 / 基本参数：电机额定参数（铭牌：功率、电压、电流、频率、转速、极数），做一次电机数据辨识。
  2. 命令源 / 设定值源：设为现场总线（p1000/p0700 相关，使控制字与设定值来自 PROFINET）。
  3. 参考转速 p2000：决定规格化换算基准。
  4. 斜坡时间 p1120/p1121：加速/减速时间。
  5. PROFINET 名称与 IP：与 TIA Portal 组态一致。
  6. 报文类型：选择标准报文 1（或 352），与 PLC 组态一致。

用 Startdrive 在 TIA Portal 里直接组态 G120，参数、报文、PN 设置一站式完成，最方便。

四、TIA Portal 组态步骤#

  1. 安装 G120 的 GSD（或用 Startdrive 自带）。
  2. 网络视图添加 G120（对应 CU 型号），连到 PLC 的 PROFINET 网络。
  3. 分配 IP 与设备名称，在线写入实物（与前述 PROFINET 设备一致，名称必须一致）。
  4. 选择报文类型（标准报文 1），系统分配过程数据的 I/Q 地址。
  5. 记下地址，例如：
    • 输出（PLC→变频器）：QW64=STW1，QW66=NSOLL_A；
    • 输入（变频器→PLC）：IW64=ZSW1，IW66=NIST_A。

五、PLC 编程示例（标准报文 1）#

// ===== 写控制字与设定值 =====
// 故障复位：在 reset 上升沿给 STW1.bit7
IF resetTrig THEN
    stw1 := 16#04FE;       // 复位位
ELSIF runCmd THEN
    stw1 := 16#047F;       // 启动（各使能 + ON）
ELSE
    stw1 := 16#047E;       // 就绪未启动
END_IF;

"QW64" := stw1;            // STW1 → 变频器

// 设定值换算：targetRpm 目标转速，refRpm 参考转速(p2000)
nsoll := REAL_TO_INT(INT_TO_REAL(targetRpm) / INT_TO_REAL(refRpm) * 16384.0);
"QW66" := nsoll;           // NSOLL_A → 变频器

// ===== 读状态字与实际值 =====
zsw1 := "IW64";            // ZSW1
running := (zsw1 AND 16#0004) <> 0;   // bit2 运行中
faulted := (zsw1 AND 16#0008) <> 0;   // bit3 故障

nist := "IW66";            // NIST_A
actualRpm := REAL_TO_INT(INT_TO_REAL(nist) / 16384.0 * INT_TO_REAL(refRpm));

启动时序建议：先持续送 16#047E 让变频器进入「就绪」，确认 ZSW1 准备就绪位 OK，再送 16#047F 启动；停车送 16#047E（OFF1 斜坡停）。

六、常见故障与排查#

现象 可能原因 排查方向
变频器不响应总线命令 STW1 bit10「由 PLC 控制」未置 1 控制字 bit10 必须为 1
命令/设定值不生效 命令源/设定值源未设为现场总线 设 p1000/p0700 为总线
PN 连接不上、设备红叉 设备名/IP 不符、报文不一致 核对名称/IP、报文类型两侧一致
转速不对 / 比例错 参考转速 p2000 与换算不符 核对 p2000 与换算公式
一启动就报 F 电机参数没调好、过流过载 重做电机辨识、查负载与接线
反转不了 STW1 bit11 或设定值符号 用 bit11 或给负设定值
停不下来/惯性停 OFF1/OFF2/OFF3 位用错 OFF1 斜坡停=bit0 清零且 bit1/2 保持 1

排查口诀#

  1. 先放控制权：STW1 bit10=1（由 PLC 控制），命令/设定值源设为总线。
  2. 再通 PN：设备名/IP/报文两侧一致，PN 连接正常。
  3. 核换算：参考转速 p2000 与设定值换算匹配。
  4. 调电机：先做电机数据辨识，再上负载。
  5. 理时序：047E 就绪 → 047F 启动 → 047E 停车；故障 bit7 上升沿复位。

七、要点速查#

环节 关键点
控制方式 PROFINET 报文（推荐），端子/USS/Modbus 备选
报文 标准报文 1（STW1+NSOLL / ZSW1+NIST）最常用
控制权 STW1 bit10=1「由 PLC 控制」必置
启停 047E 就绪 → 047F 启动 → 047E 停车
设定值 16#4000=100% 参考转速(p2000)，可负=反向
状态 ZSW1 bit2 运行、bit3 故障、bit7 报警
参数 电机辨识 + 命令源/设定值源=总线 + 报文一致

结语#

用 PROFINET 控制 G120/V20 变频器，本质上就是 PLC 周期性地写「控制字 STW1 + 速度设定值」、读「状态字 ZSW1 + 速度反馈」这四个字。只要记牢三个关键点 —— bit10 必须把控制权交给 PLC、命令/设定值源设为现场总线、设定值按 16#4000=100% 参考转速换算 —— 再配合 047E→047F 的启动时序，变频器就能被 PLC 稳定地起停调速。相比 V90 伺服的精确定位，变频器控制更简单，但报文/控制字/状态字的思路完全相通，吃透这套机制，G120、V90 乃至其他 SINAMICS 驱动都能举一反三。
